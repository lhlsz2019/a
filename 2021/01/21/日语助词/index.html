常见

1，格助词（かくじょし）

が の を に へ で と から まで より

２，接续助词（せつぞくじょし）

から ので ば と が けれども のに でも て し ながら つつ

3，并列助词（へいりつじょし）

と や か とか たり なら やら だの

4，提示助词（ていじじょし）

は も こそ でも さえ しか だって

5，副助词（ふくじょし）

ばかり だけ ほど くらい なら から まで か やら ずつ きり

6，终助词（しゆうじょし）

か ね よ な ぞ（ぜ） わ（わい） とも の さ や かしら

助词分类

格助词：表示词与词的逻辑关系

接续助词：连接词组或句子，承上启下，表因果、条件、转折、反义、举例、并列等。

并列助词：凡用于列举两个及两个以上的人或事物的助词，叫做并列助词。

提示助词：是一种添意助词，突出句子中的某一成分。

副助词：副助词接在各种词的后面，给他们增添某种意义，他是象副词一样能修饰后面用言的助词。

终助词：增加语气。
